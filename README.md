## Crypto-Wallet-System

The system will allow users to manage their crypto holdings and view current valuations built with NestJS.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Start other services (nest-cli required):

```bash
$ nest start wallet
$ nest start rate
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
